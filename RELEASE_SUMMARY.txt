INSTALLATION METHODS
--------------------
1. Homebrew (macOS/Linux)
   brew tap ananke-project/ananke
   brew install ananke

2. Direct Download
   curl -L <release-url> | tar xz
   cd ananke-v*-*
   ./install.sh

3. Custom Location
   PREFIX=~/.local ./install.sh

4. From Source
   git clone <repo> && cd ananke
   zig build

5. Rust Library
   cargo add maze

RELEASE WORKFLOW
----------------
1. Fix compilation issues (see BLOCKERS below)
2. Run: ./scripts/bump-version.sh 0.1.0
3. Update CHANGELOG.md with release notes
4. Test: ./scripts/build-release.sh
5. Commit: git commit -m "chore: bump version to 0.1.0"
6. Tag: git tag -a v0.1.0 -m "Release v0.1.0"
7. Push: git push origin main && git push origin v0.1.0
8. GitHub Actions automatically:
   - Builds for all platforms
   - Creates packages
   - Generates checksums
   - Publishes to crates.io
   - Creates Homebrew formula

VERIFICATION
------------
After release:
1. Download artifact: gh release download v0.1.0
2. Verify: ./scripts/verify-release.sh <artifact>
3. Test install: ./install.sh
4. Check binary: ananke --version
